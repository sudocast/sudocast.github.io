#19 - Infra Como Código

Posted on Tuesday, Oct 23, 2018
Olá mundo =)

Descubra neste podcast, como não ficar para trás, qual o SysAdmin que foi picado por uma aranha geneticamente modificada e se é possível fazer TDD em Infra.
